# lombok-whiskey-demo

Small project demonstrating [project Lombok](https://projectlombok.org/) features, including:

* `@Getter`
* `@Setter`
* `@EqualsAndHashCode`
* `@ToString`
* `@Data`
* `@Value`
* `@RequiredArgsConstructor`
* `@NonNull`
* `@Builder`
* `@Singular`
* `@Log*`
* `val`
* configuration system
